Серверный режим

  • НИМС первоначально задумывался как инструмент для работы в офлайне, но для подготовки сколько-нибудь больших игр необходимо координировать работу нескольких мастеров-сюжетников.
  • Для этого случая мы сделали серверный режим. На сервере находится общая база для одной игры и несколько пользователей могут подключиться к серверу и работать над игрой (по сути сервер выдает в интернет сайт).
  • На сервере используется жесткое разделение прав между мастерами.
  • Мы поднимаем сервер для вашей игры по вашему запросу.
